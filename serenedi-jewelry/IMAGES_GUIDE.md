# 添加图片指南

## 需要的图片文件

将以下图片文件添加到 `images/` 文件夹中：

### 主要图片
- `hero-jewelry.jpg` - 英雄区域展示图片 (建议尺寸: 1200x400px)
- `diamond-collection.jpg` - 戒指系列产品图片
- `jade-collection.jpg` - 项链系列产品图片
- `pearl-collection.jpg` - 手链系列产品图片
- `gold-collection.jpg` - 耳环系列产品图片
- `platinum-collection.jpg` - 手镯系列产品图片

### 品牌图片
- `logo.png` - 网站Logo (用于Open Graph分享)
- `favicon.ico` - 浏览器标签页图标 (32x32px)

### 二维码图片
- `wechat-qr.jpg` - 微信公众号/小程序二维码 (建议尺寸: 200x200px)
- `douyin-qr.jpg` - 抖音账号二维码 (建议尺寸: 200x200px)

## 图片规格建议

- **格式**: JPG/PNG/WebP
- **质量**: 80-90% (平衡文件大小和质量)
- **尺寸**: 
  - 英雄图片: 1200x400px 或更高
  - 产品图片: 400x250px 或更高
- **文件大小**: 每个图片 < 500KB

## 获取图片的方法

1. **购买/下载库存图片**:
   - Unsplash.com (免费)
   - Pexels.com (免费)
   - Shutterstock.com (付费)

2. **使用AI生成图片**:
   - Midjourney
   - DALL-E
   - Stable Diffusion

3. **拍摄实物照片**:
   - 珠宝产品照片
   - 品牌展示照片

## 添加步骤

1. 下载或创建所需的图片
2. 重命名文件以匹配上述名称
3. 将文件放入 `images/` 文件夹
4. 刷新浏览器查看效果

## 优化提示

- 使用图片压缩工具减小文件大小
- 为移动设备优化图片尺寸
- 添加适当的alt文本描述
